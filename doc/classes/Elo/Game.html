Class Elo::Game

  1. lib/elo/game.rb
Parent: Object

A Game is a collection of two Elo::Player objects and a result. Once the result is known, it propagates the new ratings to the players.

Attributes

one [R] The first Elo::Player. The result is in perspecive of this player.
result [R] The result is the result of the match. It’s a nubmer from 0 to 1 from the perspective of player :one.
two [R] The second Elo::Player.
